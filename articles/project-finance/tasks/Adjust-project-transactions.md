[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="cb092-104">Бұл тапсырма нұсқаулығында жоба бойынша транзакцияны реттеу жолы көрсетілген.</span><span class="sxs-lookup"><span data-stu-id="cb092-104">This task guide shows you how to adjust a transaction on a project.</span></span> <span data-ttu-id="cb092-105">Алайда, жоба транзакцияларын бір-бірлеп реттеуге немесе барлық транзакциялар тізімінен таңдауға болады.</span><span class="sxs-lookup"><span data-stu-id="cb092-105">However, you can adjust project transactions one at a time, or select from a list of all project transactions.</span></span> <span data-ttu-id="cb092-106">Бұл тапсырманы әдетте жоба супервайзері орындайды.</span><span class="sxs-lookup"><span data-stu-id="cb092-106">This task is usually completed by the Project supervisor.</span></span> <span data-ttu-id="cb092-107">Бұл тапсырма нұсқаулығында USSI деректер жиынтығы пайдаланылады.</span><span class="sxs-lookup"><span data-stu-id="cb092-107">This task guide uses the USSI data set.</span></span>

1. <span data-ttu-id="cb092-108">Жобаларды басқару және есепке алу > Жобалар > Барлық жобалар тармағына өтіңіз.</span><span class="sxs-lookup"><span data-stu-id="cb092-108">Go to Project management and accounting > Projects > All projects.</span></span> 
2. <span data-ttu-id="cb092-109">Тізімде таңдалған жолды белгілеңіз.</span><span class="sxs-lookup"><span data-stu-id="cb092-109">In the list, mark the selected row.</span></span> 
3. <span data-ttu-id="cb092-110">"Транзакцияларды реттеу" пәрменін басыңыз.</span><span class="sxs-lookup"><span data-stu-id="cb092-110">Click Adjust transactions.</span></span> 
4. <span data-ttu-id="cb092-111">"Таңдау" түймесін басыңыз.</span><span class="sxs-lookup"><span data-stu-id="cb092-111">Click Select.</span></span> 
5. <span data-ttu-id="cb092-112">Сағат өрісінде "Иә" таңдаңыз.</span><span class="sxs-lookup"><span data-stu-id="cb092-112">Select Yes in the Hour field.</span></span> 
6. <span data-ttu-id="cb092-113">Күн өрісіне күнді енгізіңіз.</span><span class="sxs-lookup"><span data-stu-id="cb092-113">In the To date field, enter a date.</span></span> 
7. <span data-ttu-id="cb092-114">OK түймешігін басыңыз.</span><span class="sxs-lookup"><span data-stu-id="cb092-114">Click OK.</span></span> 
8. <span data-ttu-id="cb092-115">Тізімде таңдалған жолды белгілеңіз.</span><span class="sxs-lookup"><span data-stu-id="cb092-115">In the list, mark the selected row.</span></span> 
9. <span data-ttu-id="cb092-116">"Реттеу" түймесін басыңыз.</span><span class="sxs-lookup"><span data-stu-id="cb092-116">Click Adjust.</span></span> 
10. <span data-ttu-id="cb092-117">Тізімде таңдалған жолды белгілеңіз.</span><span class="sxs-lookup"><span data-stu-id="cb092-117">In the list, mark the selected row.</span></span> 
11. <span data-ttu-id="cb092-118">Жолақ сипаты өрісіне мәнді енгізіңіз немесе таңдаңыз.</span><span class="sxs-lookup"><span data-stu-id="cb092-118">In the Line property field, enter or select a value.</span></span> 
12. <span data-ttu-id="cb092-119">OK түймешігін басыңыз.</span><span class="sxs-lookup"><span data-stu-id="cb092-119">Click OK.</span></span> 
13. <span data-ttu-id="cb092-120">Тізімде таңдалған жолды белгілеңіз.</span><span class="sxs-lookup"><span data-stu-id="cb092-120">In the list, mark the selected row.</span></span> 
14. <span data-ttu-id="cb092-121">Шығын өрісіне сан енгізіңіз.</span><span class="sxs-lookup"><span data-stu-id="cb092-121">In the Cost price field, enter a number.</span></span> 
15. <span data-ttu-id="cb092-122">"Жариялау" түймесін басыңыз.</span><span class="sxs-lookup"><span data-stu-id="cb092-122">Click Post.</span></span>
