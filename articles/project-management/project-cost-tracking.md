<span data-ttu-id="2b907-148">Жиынтық тапсырмалар немесе контейнер тапсырмалары бойынша еңбек шығындарын қайта жобалауға болады.</span><span class="sxs-lookup"><span data-stu-id="2b907-148">You can reproject labor costs on summary tasks or container tasks.</span></span> <span data-ttu-id="2b907-149">Дегенмен жобаның жиынтық тапсырмасы үшін **Жоба** бетіндегі **Бақылау** қойыншасындағы еңбек шығындарын тікелей қайта жобалай алмайсыз.</span><span class="sxs-lookup"><span data-stu-id="2b907-149">However, you can't directly reproject labor cost on a summary project task on the **Tracking** tab on the **Project** page.</span></span> <span data-ttu-id="2b907-150">Соңғы түйіндерінің тапсырмалары сияқты жиынтық және контейнер тапсырмаларын **Талпынысты бақылау** көрінісі арқылы қайта жобалауға болады.</span><span class="sxs-lookup"><span data-stu-id="2b907-150">Similar to leaf node tasks, reprojecting summary and container tasks can be done using the **Effort Tracking** view.</span></span> <span data-ttu-id="2b907-151">Бұл көріністе жиынтық тапсырма бойынша қалған құнын қайта есептеуді тудыратын жиынтық тапсырма бойынша қалған талпынысты қайта жобалай аласыз.</span><span class="sxs-lookup"><span data-stu-id="2b907-151">In this view, you can reproject the remaining effort on a summary task causing a recalculation of the remaining cost on the summary task.</span></span> <span data-ttu-id="2b907-152">Төменде мұның қалай жұмыс істейтіні сипатталған.</span><span class="sxs-lookup"><span data-stu-id="2b907-152">The following is a description of how this works.</span></span>

2. <span data-ttu-id="2b907-156">Жиынтық тапсырма бойынша қалған талпыныстың жаңа мәніне сүйене отырып, жүйе **Шығындарды қадағалау** көрінісіндегі қалған құнды есептейді.</span><span class="sxs-lookup"><span data-stu-id="2b907-156">Based on the new value for remaining effort on the summary task, the system calculates the remaining cost on the **Cost Tracking** view.</span></span> <span data-ttu-id="2b907-157">Қалған талпынысқа негізделген қалған құнын есептеу үшін жүйе алдымен жиынтық тапсырма бойынша бір сағаттық талпыныстың орташа құнын жоспарланған шығын немесе жоспарланған талпыныс ретінде есептейді.</span><span class="sxs-lookup"><span data-stu-id="2b907-157">To calculate the remaining cost based on the remaining effort, the system first calculates the average cost of one hour of effort on the summary task as planned cost or planned effort.</span></span> <span data-ttu-id="2b907-158">Сағатына жұмсалатын орташа шығындар жиынтық тапсырма бойынша жаңадан жобаланған қалған талпыныстың құнын есептеу үшін қолданылады.</span><span class="sxs-lookup"><span data-stu-id="2b907-158">The average cost per hour is used to calculate the cost of the newly projected remaining effort on the summary task.</span></span>
