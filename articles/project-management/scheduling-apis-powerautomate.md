# <a name="use-project-schedule-apis-with-power-automate"></a>Жоба кестесі API интерфейстерін пайдаланыңыз Power Automate

_**Келесіге қолданылады:** ресурс/биржадан тыс негіздегі сценарийлерге арналған Project Operations, жеңілдетілген орналастыру - проформа-шотын ұсыну мәмілесі_

Төменде осы тақырыптағы үлгі ағынында құжатталған қадамдардың толық тізімі берілген:

## <a name="assumptions"></a>Болжамдар

Бұл тақырып сіздің негізгі біліміңіз бар деп болжайды Dataverse платформа, бұлттық ағындар және Жоба кестесі қолданбалы бағдарламалау интерфейсі (API). Қосымша ақпаратты мына жерден қараңыз [Анықтамалар](#references) осы тақырыптың кейінгі бөлімі.

## <a name="create-a-flow"></a>Ағын жасау

### <a name="select-an-environment"></a>Орта таңдау

жасай аласыз Power Automate ортаңызда ағын.

1. Бару<https://flow.microsoft.com>, және жүйеге кіру үшін әкімші тіркелгі деректерін пайдаланыңыз.
2. Жоғарғы оң жақ бұрышта таңдаңыз **Қоршаған орталар**.
3. Тізімде ортаны таңдаңыз Dynamics 365 Project Operations орнатылған.

### <a name="create-a-solution"></a>Шешім жасау

жасау үшін мына қадамдарды орындаңыз [шешімді білетін ағын](/power-automate/overview-solution-flows). Шешімді ескеретін ағын жасау арқылы оны кейінірек пайдалану үшін ағынды оңай экспорттай аласыз.

1. Шарлау тақтасында таңдаңыз **Шешімдер**.
2. Үстінде **Шешімдер** бет, таңдаңыз **Жаңа шешім**.
3. Ішінде **Жаңа шешім** тілқатысу терезесінде қажетті өрістерді орнатыңыз, содан кейін таңдаңыз **Жасау**.

## <a name="step-1-create-a-powerapps-trigger"></a><a id="1"></a> 1-қадам: a PowerApps триггер

1. Үстінде **Шешімдер** бетінде жасалған шешімді таңдаңыз, содан кейін таңдаңыз **Жаңа**.
2. Сол жақ аумақта таңдаңыз **Бұлт ағып жатыр** \> **Автоматтандыру** \> **Бұлт ағыны** \> **Лезде**.
3. Ішінде **Ағын атауы** өріс, енгізіңіз **API демо ағынын жоспарлау**.
4. Ішінде **Бұл ағынды іске қосу жолын таңдаңыз** тізім, таңдаңыз **Power Apps**. Сіз жасаған кезде а Power Apps триггер, логика автор ретінде сізге байланысты. Бұл тақырыпта сынақ мақсаттары үшін енгізу параметрлерін бос қалдырыңыз.
5. **Жасау** пәрменін таңдаңыз.

## <a name="step-2-create-a-project"></a><a id="2"></a>2-қадам: Жоба жасау

Үлгі жобаны жасау үшін мына қадамдарды орындаңыз.

1. Сіз жасаған ағында таңдаңыз **Жаңа қадам**.

    ![Жаңа қадам қосу.](media/newstep.png)

2. Ішінде **Операцияны таңдаңыз** тілқатысу терезесінің іздеу өрісіне енгізіңіз **шектелмеген әрекетті орындау**. Содан кейін, бойынша **Әрекеттер** қойындысында нәтижелер тізімінен әрекетті таңдаңыз.

    ![Операцияны таңдау.](media/chooseactiontab.png)

3. Жаңа қадамда эллипсті таңдаңыз (**...**), содан кейін таңдаңыз **Атын өзгерту**.

![Қадамның атын өзгерту.](media/renamestep.png)

4. Қадамның атын өзгерту **Жоба жасау**.
5. Ішінде **Әрекет атауы** өрісін таңдаңыз **msdyn\_ CreateProjectV1**.
6. Астында **msdyn\_ пән** өрісін таңдаңыз **Динамикалық мазмұнды қосыңыз**.
7. Үстінде **Өрнек** қойындысының функция өрісіне енгізіңіз **Жоба атауы - utcNow()**.
8. **OK** пәрменін таңдаңыз.

## <a name="step-3-initialize-a-variable-for-the-team-member"></a><a id="3"></a> 3-қадам: Топ мүшесі үшін айнымалы мәнді инициализациялаңыз

1. Ағында таңдаңыз **Жаңа қадам**.
2. Ішінде **Операцияны таңдаңыз** тілқатысу терезесінің іздеу өрісіне енгізіңіз **айнымалыны инициализациялау**. Содан кейін, бойынша **Әрекеттер** қойындысында нәтижелер тізімінен әрекетті таңдаңыз.
3. Жаңа қадамда эллипсті таңдаңыз (**...**), содан кейін таңдаңыз **Атын өзгерту**.
4. Қадамның атын өзгерту **Топ мүшесі**.
5. Ішінде **Аты** өріс, енгізіңіз **TeammemberAction**.
6. Ішінде **Түр** өрісін таңдаңыз **Жол**.
7. Ішінде **Мән** өріс, енгізіңіз **msdyn\_ CreateTeammemberV1**.

## <a name="step-4-create-a-generic-team-member"></a><a id="4"></a> 4-қадам: Жалпы топ мүшесін жасаңыз

1. Ағында таңдаңыз **Жаңа қадам**.
2. Ішінде **Операцияны таңдаңыз** тілқатысу терезесінің іздеу өрісіне енгізіңіз **шектелмеген әрекетті орындау**. Содан кейін, бойынша **Әрекеттер** қойындысында нәтижелер тізімінен әрекетті таңдаңыз.
3. Жаңа қадамда эллипсті таңдаңыз (**...**), содан кейін таңдаңыз **Атын өзгерту**.
4. Қадамның атын өзгерту **Топ мүшесін жасау**.
5. үшін **Әрекет атауы** өрісін таңдаңыз **TeammemberAction** ішінде **Динамикалық мазмұн** диалогтық терезе.
6. Ішінде **Әрекет параметрлері** өрісіне келесі параметр ақпаратын енгізіңіз.

    ```
    {
        "TeamMember": {
            "@@odata.type": "Microsoft.Dynamics.CRM.msdyn_projectteam",
            "msdyn_projectteamid": "@{guid()}",
            "msdyn_project@odata.bind": "/msdyn_projects(@{outputs('Create_Project')?['body/ProjectId']})",
            "msdyn_name": "ScheduleAPIDemoTM1"
        }
    } 
    ```

    Мұнда параметрлердің түсіндірмесі берілген:

    - **\@\@ odata.type** – Нысан атауы. Мысалы, енгізіңіз **"Microsoft.Dynamics.CRM.msdyn\_ жоба командасы»**.
    - **msdyn\_ жобалық команда** – Жоба командасының идентификаторының негізгі кілті. Мән жаһандық бірегей идентификатор (GUID) өрнегі болып табылады.   Идентификатор өрнек қойындысынан жасалады.

    - **msdyn\_ жоба\@ odata.bind** – Иесі жобаның жоба идентификаторы. Мән «Жоба жасау» қадамының жауабынан келетін динамикалық мазмұн болады. Толық жолды енгізіп, жақшалар арасына динамикалық мазмұнды қосқаныңызға көз жеткізіңіз. Тырнақша қажет. Мысалы, енгізіңіз **"/msdyn\_ жобалар (ДИНАМИЯЛЫҚ МАЗМҰН ҚОСУ)»**.
    - **msdyn\_ аты** – Топ мүшесінің аты-жөні. Мысалы, енгізіңіз **"ScheduleAPIDemoTM1"**.

## <a name="step-5-create-an-operation-set"></a><a id="5"></a> 5-қадам: Әрекеттер жинағын жасаңыз

1. Ағында таңдаңыз **Жаңа қадам**.
2. Ішінде **Операцияны таңдаңыз** тілқатысу терезесінің іздеу өрісіне енгізіңіз **шектелмеген әрекетті орындау**. Содан кейін, бойынша **Әрекеттер** қойындысында нәтижелер тізімінен әрекетті таңдаңыз.
3. Жаңа қадамда эллипсті таңдаңыз (**...**), содан кейін таңдаңыз **Атын өзгерту**.
4. Қадамның атын өзгерту **Операциялар жинағын жасаңыз**.
5. Ішінде **Әрекет атауы** өрісін таңдаңыз **msdyn\_ CreateOperationSetV1** Dataverse реттелетін әрекет.
6. Ішінде **Сипаттама** өріс, енгізіңіз **ScheduleAPIDemoOperationSet**.
7. Ішінде **Жоба** өріс, енгізіңіз **/msdyn\_ жобалар(**.
8. Ішінде **Динамикалық мазмұн** диалогтық терезеде таңдаңыз **msdyn\_ CreateProjectV1Response ProjectId**.
9. Ішінде **Жоба** өріс, енгізіңіз **)**.

## <a name="step-6-create-a-project-bucket"></a><a id="6"></a> 6-қадам: жоба шелегін жасаңыз

1. Ағында таңдаңыз **Жаңа қадам**.
2. Ішінде **Операцияны таңдаңыз** тілқатысу терезесінің іздеу өрісіне енгізіңіз **жаңа жолды қосыңыз**. Содан кейін, бойынша **Әрекеттер** қойындысында нәтижелер тізімінен әрекетті таңдаңыз.
3. Жаңа қадамда эллипсті таңдаңыз (**...**), содан кейін таңдаңыз **Атын өзгерту**.
4. Қадамның атын өзгерту **Шелек жасау**.
5. Ішінде **Кесте атауы** өрісін таңдаңыз **Жоба шелектері**.
6. Ішінде **Аты** өріс, енгізіңіз **ScheduleAPIDemoBucket1**.
7. үшін **Жоба** өрісін таңдаңыз **msdyn\_ CreateProjectV1Response ProjectId** ішінде **Динамикалық мазмұн** диалогтық терезе.

## <a name="step-7-initialize-a-variable-for-the-link-status"></a><a id="7"></a> 7-қадам: Сілтеме күйі үшін айнымалы мәнді инициализациялаңыз

1. Ағында таңдаңыз **Жаңа қадам**.
2. Ішінде **Операцияны таңдаңыз** тілқатысу терезесінің іздеу өрісіне енгізіңіз **айнымалыны инициализациялау**. Содан кейін, бойынша **Әрекеттер** қойындысында нәтижелер тізімінен әрекетті таңдаңыз.
3. Жаңа қадамда эллипсті таңдаңыз (**...**), содан кейін таңдаңыз **Атын өзгерту**.
4. Қадамның атын өзгерту **Сілтеме күйін бастау**.

6. Ішінде **Түр** өрісін таңдаңыз **бүтін сан**.
7. Ішінде **Мән** өріс, енгізіңіз **192350000**.

## <a name="step-8-initialize-a-variable-for-the-number-of-tasks"></a><a id="8"></a> 8-қадам: Тапсырмалар саны үшін айнымалы мәнді инициализациялаңыз

5. Ішінде **Аты** өріс, енгізіңіз **тапсырмалар саны**.

## <a name="step-9-initialize-a-variable-for-the-project-task-id"></a><a id="9"></a> 9-қадам: жоба тапсырмасының идентификаторы үшін айнымалы мәнді инициализациялаңыз

3. Жаңа қадамда эллипсті таңдаңыз (**...**), содан кейін таңдаңыз **Атын өзгерту**.
4. Қадамның атын өзгерту **ProjectTaskID іске қосыңыз**.
5. Ішінде **Аты** өріс, енгізіңіз **тапсырмалар саны**.

## <a name="step-10-do-until"></a><a id="10"></a> 10-қадам: дейін орындаңыз

1. Ағында таңдаңыз **Жаңа қадам**.
2. Ішінде **Операцияны таңдаңыз** тілқатысу терезесінің іздеу өрісіне енгізіңіз **дейін жасаңыз**. Содан кейін, бойынша **Әрекеттер** қойындысында нәтижелер тізімінен әрекетті таңдаңыз.
3. Шартты оператордағы бірінші мәнді орнатыңыз **тапсырмалар саны** -дан айнымалы **Динамикалық мазмұн** диалогтық терезе.
4. Шартты орнатыңыз **теңінен аз**.
5. Шартты оператордағы екінші мәнді орнатыңыз **0**.

## <a name="step-11-set-a-project-task"></a><a id="11"></a> 11-қадам: жоба тапсырмасын орнатыңыз

1. Ағында таңдаңыз **Жаңа қадам**.
2. Ішінде **Операцияны таңдаңыз** тілқатысу терезесінің іздеу өрісіне енгізіңіз **айнымалыны орнату**. Содан кейін, бойынша **Әрекеттер** қойындысында нәтижелер тізімінен әрекетті таңдаңыз.
3. Жаңа қадамда эллипсті таңдаңыз (**...**), содан кейін таңдаңыз **Атын өзгерту**.
4. Қадамның атын өзгерту **Жоба тапсырмасын орнату**.
5. Ішінде **Аты** өрісін таңдаңыз **msdyn\_ жобалық тапсырма**.

## <a name="step-12-create-a-project-task"></a><a id="12"></a> 12-қадам: Жоба тапсырмасын жасаңыз

Ағымдағы жобаға және сіз жасаған жоба шелегіне жататын бірегей идентификаторы бар жоба тапсырмасын жасау үшін мына қадамдарды орындаңыз.

1. Ағында таңдаңыз **Жаңа қадам**.
2. Ішінде **Операцияны таңдаңыз** тілқатысу терезесінің іздеу өрісіне енгізіңіз **шектелмеген әрекетті орындау**. Содан кейін, бойынша **Әрекеттер** қойындысында нәтижелер тізімінен әрекетті таңдаңыз.
3. Қадамда эллипсті таңдаңыз (**...**), содан кейін таңдаңыз **Атын өзгерту**.
4. Қадамның атын өзгерту **Жоба тапсырмасын жасау**.
5. Ішінде **Әрекет атауы** өрісін таңдаңыз **msdyn\_ PssCreateV1**.
6. Ішінде **Нысан** өрісіне келесі параметр ақпаратын енгізіңіз.

    ```
    {
        "@@odata.type": "Microsoft.Dynamics.CRM.msdyn_projecttask",
        "msdyn_projecttaskid": "@{variables('msdyn_projecttaskid')}",
        "msdyn_project@odata.bind": "/msdyn_projects(@{outputs('Create_Project')?['body/ProjectId']})",
        "msdyn_subject": "ScheduleAPIDemoTask1",
        "msdyn_projectbucket@odata.bind": "/msdyn_projectbuckets(@{outputs('Create_Project_Buckets')?['body/msdyn_projectbucketid']})",
        "msdyn_start": "@{addDays(utcNow(), 1)}",
        "msdyn_scheduledstart": "@{utcNow()}",
        "msdyn_scheduledend": "@{addDays(utcNow(), 5)}",
        "msdyn_LinkStatus": "192350000"
    }
    ```

    Мұнда параметрлердің түсіндірмесі берілген:

    - **\@\@ odata.type** – Нысан атауы. Мысалы, енгізіңіз **"Microsoft.Dynamics.CRM.msdyn\_ жобалық тапсырма»**.
    - **msdyn\_ жобалық тапсырма** – Тапсырманың бірегей идентификаторы. Мән келесіден динамикалық айнымалыға орнатылуы керек **msdyn\_ жобалық тапсырма**.
    - **msdyn\_ жоба\@ odata.bind** – Иесі жобаның жоба идентификаторы. Мән «Жоба жасау» қадамының жауабынан келетін динамикалық мазмұн болады. Толық жолды енгізіп, жақшалар арасына динамикалық мазмұнды қосқаныңызға көз жеткізіңіз. Тырнақша қажет. Мысалы, енгізіңіз **"/msdyn\_ жобалар (ДИНАМИЯЛЫҚ МАЗМҰН ҚОСУ)»**.
    - **msdyn\_ пән** – Кез келген тапсырма атауы.
    - **msdyn\_ жобалық шелек\@ odata.bind** – Тапсырмаларды қамтитын жоба шелегі. Мән «Шөміш жасау» қадамының жауабынан келетін динамикалық мазмұн болады. Толық жолды енгізіп, жақшалар арасына динамикалық мазмұнды қосқаныңызға көз жеткізіңіз. Тырнақша қажет. Мысалы, енгізіңіз **"/msdyn\_ жобалар шелектері (ДИНАМИКАЛЫҚ МАЗМҰН ҚОСУ)"**.
    - **msdyn\_ бастау** – Басталу күні үшін динамикалық мазмұн. Мысалы, ертең ретінде көрсетіледі **"addDays(utcNow(), 1)"**.
    - **msdyn\_ жоспарланған бастау** – Жоспарланған басталу күні. Мысалы, ертең ретінде көрсетіледі **"addDays(utcNow(), 1)"**.
    - **msdyn\_ кесте** – Жоспарланған аяқталу күні. Болашақта күнді таңдаңыз. Мысалы, көрсетіңіз **"addDays(utcNow(), 5)"**.
    - **msdyn\_ Сілтеме күйі** – Сілтеме күйі. Мысалы, енгізіңіз **"192350000"**.

7. үшін **OperationSetId** өрісін таңдаңыз **msdyn\_ CreateOperationSetV1Response** ішінде **Динамикалық мазмұн** диалогтық терезе.
