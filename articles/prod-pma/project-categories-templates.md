# <a name="synchronize-project-expense-categories-between-finance-and-operations-and-project-service-automation"></a>Жобаның шығыс санаттарын Finance and Operations және Project Service Automation арасында синхрондаңыз

[!include[banner](../includes/banner.md)]

Бұл тақырыпта жобаның шығыс санаттарын Dynamics 365 Finance және Dynamics 365 Project Service Automation арасында синхрондау үшін пайдаланылатын үлгілер мен негізгі тапсырмалар сипатталады.

> [!NOTE]
> - Жоба тапсырмаларын біріктіру, шығындар транзакциясының санаттары, сағаттық болжамдар, шығын болжамдары және функционалды құлыптау 8.0 нұсқасында қолжетімді.
> - Нақты көрсеткіштерді біріктіру 8.0.1 немесе одан кейінгі нұсқада қолжетімді.
> - Егер Enterprise Edition 7.3.0 нұсқасын пайдалансаңыз, 4132657 білім қоры мен 4132660 білім қорын орнатқаннан кейін, үлгілерді жоба тапсырмаларын, шығыс транзакция санаттарын, сағаттық болжамдарды, шығын болжамдарын және нақты көрсеткіштерді біріктіру және функционалды құлыптауды конфигурациялау үшін пайдалана аласыз. Бухгалтерлік шоттар бойынша таратуларды қалпына келтіру қажет, сондай-ақ 4131710 білім қорын орнатуды ұсынамыз.

## <a name="data-flow-for-project-service-automation-and-finance"></a>Project Service Automation және Finance бағдарламасына арналған деректер ағыны

Project Service Automation және Finance біріктіру шешімі Project Service Automation және Finance даналары ішінде деректерді синхрондау үшін деректерді біріктіру мүмкіндігін пайдаланады. Деректерді біріктіру функциясымен қолжетімді болатын біріктіру үлгілері жобаның шығыс транзакциясы санаттары туралы деректер ағынын Finance және Project Service Automation арасында қосады.

Егер жобаның шығыс санаттары Finance жүйесінде игерілсе, біріктіру ағыны Finance жүйесінен Project Service Automation бағдарламасына бірінші орында тұрады. Жобаның шығыс санаттарының біріктіру идентификаторлары Project Service Automation бағдарламасынан Finance жүйесіге синхрондалу арқылы жаңартылады.

Егер жобаның шығыс санаттары Project Service Automation бағдарламасында игерілсе, біріктіру ағыны Project Service Automation бағдарламасынан Finance жүйесіне бірінші орында тұрады. Жоба санаттары Project Service Automation бағдарламасынан синхрондалудан бұрын Finance жүйесінде конфигурациялануы қажет. Содан соң Finance жүйесінен қайтадан Project Service Automation бағдарламасына синхрондап, содан соң тағы да Project Service Automation бағдарламасынан Finance жүйесіне синхрондаңыз. Осылайша, сіз санаттар бір-бірімен байланысты екеніне және ешқандай көшірмелер жасалмайтынына кепілдік бересіз.

> [!NOTE]
> Әдетте жобаның шығыс санаттары Finance жүйесінде игеріледі. Дегенмен, егер олай болмаса немесе егер шығыс санаттары Project Service Automation бағдарламасында жасалса, жобаның шығыс транзакция санаттары (PSA бағдарламасынан Fin және Ops бағдарламасына) үлгісін пайдалану арқылы алдымен синхрондау қажет. Содан соң жобаның шығыс транзакция санаттарының (Fin және Ops бағдарламасынан PSA бағдарламасына) үлгісі арқылы синхрондаңыз. Содан кейін Project Service Automation бағдарламасынан Finance бағдарламасына тағы бір рет синхрондауды жүргізу қажет.
>
> Егер алдымен Project Service Automation бағдарламасынан синхрондайтын болсаңыз, синхрондалу жүргізілмес бұрын Finance жүйесінде келесі талаптар орындалуы қажет:
>
> - Project Service Automation бағдарламасында орнатылған жоба санатына сәйкес келетін ортақ пайдаланылатын санат болуы қажет және ол **Жоба** үшін де, сондай-ақ **Шығыс** үшін де қосулы болуы қажет.
> - Біріктірілуі тиіс әрбір Finance заңды нысаны үшін келесі жоба санаттары болуы қажет:
>
>     - **Жоба санаты** бар. 
>     - **Шығыста пайдалану** қосылған.
>     - **Журналда белсенді** қосылған.
>     - **Транзакция түрі** **Шығыс** параметріне орнатылған.

Келесі суретте деректердің Project Service Automation және Finance арасында синхрондалу жолы көрсетілген.

## <a name="project-expense-category-synchronization-from-finance-to-project-service-automation"></a>Жобаның шығыс санатын Finance жүйесінен Project Service Automation бағдарламасына синхрондау

### <a name="template-and-task"></a>Үлгі және тапсырма

Үлгіге қатынасу мақсатында, жалпы үлгілерді таңдау үшін Microsoft Power Apps басқару орталығында **Жобалар**, содан кейін оң жақ жоғарғы бұрышта **Жаңа жоба** пәрменін таңдаңыз.

Келесі үлгі және негізгі тапсырма жобаның шығыс санаттарын Finance жүйесінен Project Service Automation бағдарламасына синхрондау үшін пайдаланылады:

- **Деректерді біріктіру функциясындағы үлгі атауы:** жобаның шығыс транзакция санаттары (Fin және Ops бағдарламасынан PSA бағдарламасына)
- **Жобадағы тапсырма атауы:** PSA бағдарламасына синхрондау санаттары

### <a name="entity-set"></a>Нысан жиынтығы

| Қаржы                           | Project Service Automation |
|-----------------------------------|----------------------------|
| Санаттарға арналған біріктіру нысаны | Транзакция санаттары     |

### <a name="entity-flow"></a>Нысан ағыны

Жобаның шығыс санаттары Finance бағдарламасында басқарылады және олар Project Service Automation бағдарламасына транзакция санаттары ретінде синхрондалады.

### <a name="power-query"></a>Power Query

Project Service Automation бағдарламасына синхрондау кезінде транзакция санатында шот ұсыну түрін орнату үшін Excel бағдарламасына арналған Microsoft Power Query бағдарламасын пайдалану қажет. Жобаның шығыс транзакция санаттары (Fin және Ops бағдарламасынан PSA бағдарламасына) үлгісі әдепкі баған мен салыстыруды қамтамасыз етеді. Егер жеке үлгіңізді жасасаңыз, Power Query бағдарламасында шартты бағанды қосу қажет. Осы қадамдарды орындаңыз.

1. Жобаның шығыс транзакция санаттарының (Fin және Ops бағдарламаларынан PSA бағдарламасына) үлгісінде жобаның шығыс санаттарыны салыстырылуын ашу үшін көрсеткіні басыңыз.
2. Power Query бағдарламасын ашу үшін **Кеңейтілген сұрау және сүзгілеу** сілтемесін басыңыз.
2. **Шартты бағанды қосу** опциясын таңдаңыз.
3. Жаңа бағанға арналған **BillingType** сияқты атауды енгізіңіз.
4. Келесі шартты енгізіңіз: **егер САНАТ ИДЕНТИФИКАТОРЫ нөлге тең болмаса, онда 19235001, нөлге тең**.
5. Бағанда **ОК** түймешігін басыңыз.
6. Осы жаңа бағанды салыстыру бетінде салыстырыңыз.

Келесі суретте деректерді біріктіру қызметіндегі үлгі тапсырмасын салыстыру мысалы көрсетіледі. Салыстыру Finance жүйесінен Project Service Automation жүйесіне синхрондалатын өріс ақпаратын көрсетеді.

## <a name="project-expense-category-synchronization-from-project-service-automation-to-finance"></a>Жобаның шығыс санатын Project Service Automation бағдарламасынан Finance жүйесіне синхрондау

### <a name="template-and-task"></a>Үлгі және тапсырма

Келесі үлгі және негізгі тапсырма жобаның шығыс санаттарын Project Service Automation бағдарламасынан Finance жүйесіне синхрондау үшін пайдаланылады:

- **Деректерді біріктіру функциясындағы үлгі атауы:** жобаның шығыс транзакция санаттары (PSA бағдарламасынан Fin және Ops бағдарламасына)
- **Жобадағы тапсырма атауы:** Fin Ops бағдарламасына синхрондау санаттары

### <a name="entity-set"></a>Нысан жиынтығы

| Project Service Automation | Қаржы                           |
|----------------------------|-----------------------------------|
| Транзакция санаттары     | Санаттарға арналған біріктіру нысаны |

### <a name="entity-flow"></a>Нысан ағыны

Жобаның шығыс санаттары Finance бағдарламасында басқарылады және олар Project Service Automation бағдарламасына транзакция санаттары ретінде синхрондалады. Finance жүйесіне қайта синхрондау Project Service Automation бағдарламасының біріктіру идентификаторымен Finance жүйесінде жоба санатын жаңартады.

Келесі суретте деректерді біріктіру қызметіндегі үлгі тапсырмасын салыстыру мысалы көрсетіледі.
