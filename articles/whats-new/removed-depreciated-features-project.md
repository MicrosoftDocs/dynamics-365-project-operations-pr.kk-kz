# <a name="removed-or-deprecated-features-in-dynamics-365-project-operations"></a>Dynamics 365 Project Operations бағдарламасындағы жойылған немесе ұсынылмайтын мүмкіндіктер

_**Келесіге қолданылады:** ресурс/биржадан тыс негіздегі сценарийлерге арналған Project Operations, жеңілдетілген орналастыру - проформа-шотын ұсыну мәмілесі және қосалқы/өндіріске негізделген сценарийлеріне арналған Project Operations_

Бұл мақалада Dynamics 365 Project Operations бағдарламасында жойылған немесе жою жоспарланған мүмкіндіктер сипатталған.

- *Жойылған* мүмкіндік өнім ішінде бұдан былай қолжетімді емес.
- *Ұсынылмаған* функциясы белсенді әзірленуде емес және кейінгі жаңартуда жойылуы мүмкін.

Бұл тізім жоспарлауыңыз үшін осы жою әрекеттері мен қарсылықтарды қарастыруға көмектесуге арналған.

> [!NOTE]
> Қаржы және операцияларға арналған бағдарламалардағы нысандар туралы толық ақпаратты [**Техникалық анықтамалық есептер**](/dynamics/s-e/global/axtechrefrep_61) бөлімінен табуға болады. Қаржы және операцияларға арналған бағдарламалардың әрбір нұсқасында өзгертілген немесе жойылған нысандар туралы толығырақ білу үшін осы есептердің әртүрлі нұсқасын салыстыруға болады.

## <a name="features-removed-or-deprecated-in-the-project-operations-march-2022-release"></a>Project Operations бағдарламасының 2022 жылғы наурыздағы шығарылымында жойылған немесе ұсынылмаған мүмкіндіктер

### <a name="project-management-and-accounting-always-create-adjustment-transaction-parameter"></a>Жобаны басқару және есеп бойынша «Реттеу транзакциясын әрқашан жасау» параметрі

| &nbsp; | &nbsp; |
|--------|--------|
| **Ұсынбау/жою себебі** | Реттеу транзакциялары тексеру мақсаттары үшін қажет. Депрекациядан кейін бұл параметр жасырылады. Жүйе әрқашан реттеу транзакцияларын жасайды, бұл қазіргі уақытта параметр **Иә** күйіне орнатылған кездегідей болады. |
| **Басқа мүмкіндікпен ауыстырылды ма?** | No |
| **Өнімнің әсерге ұшыраған аймақтары** | Қолданба |
| **Орналастыру опциясы** | Өндіріс/қосалқы тапсырысы сценарийлеріне арналған Project Operations бағдарламасы |
| **Күйі** | Ұсынылмаған: 2023 жылдың 1 наурызына қарай біз параметрді жасырып, жүйе әрекетін өзгертеміз, осылайша реттеу транзакциялары әрқашан жасалатын болады. |

### <a name="project-management-and-accounting-use-adjustment-date-as-new-project-date-parameter"></a>Жобаны басқару және есеп бойынша «Жаңа жоба күні ретінде реттеу күнін пайдалану» параметрі

| &nbsp; | &nbsp; |
|--------|--------|
| **Ұсынбау/жою себебі** | Бұл параметр бастапқыда қаржы кезеңі жабылған кезде реттеуге рұқсат беру үшін пайдаланылған. Дегенмен ол енді қажет емес, себебі транзакцияның есептік күні конфигурацияланған болса, ашық кезеңнің бірінші күнін өзгертуге болады. Жоба күнін өзгертуге болмайды, себебі ол транзакция жасалған күнді білдіреді. |
| **Басқа мүмкіндікпен ауыстырылды ма?** | No |
| **Өнімнің әсерге ұшыраған аймақтары** | Қолданба |
| **Орналастыру опциясы** | Өндіріс/қосалқы тапсырысы сценарийлеріне арналған Project Operations бағдарламасы |
| **Күйі** | Ұсынылмаған: 2023 жылдың 1 наурызына қарай біз параметрді жасырып, жүйе әрекетін өзгертеміз, осылайша реттеулер кезінде жоба күні ешқашан өзгермейтін болады. |

### <a name="resource-request-workflow-in-project-operations-for-stockedproduction-based-scenarios"></a>Қосалқы/өндіріске негізделген сценарийлерге арналған Project Operations бағдарламасындағы ресурс сұрауының жұмыс ағыны

| &nbsp; | &nbsp; |
|--------|--------|
| **Ұсынбау/жою себебі** | Пайдалану деңгейінің төмендігіне және транзакция көлемінің шектеулеріне байланысты ұсынылмаған. |
| **Басқа мүмкіндікпен ауыстырылды ма?** | No |
| **Өнімнің әсерге ұшыраған аймақтары** | Қолданба |
| **Орналастыру опциясы** | Өндіріс/қосалқы тапсырысы сценарийлеріне арналған Project Operations бағдарламасы |
| **Күйі** | Ұсынылмаған: 2023 жылдың 1 наурызына қарай жұмыс ағынының көмегімен жобаға арналған ресурстарды сұрау опциясын өшіреміз. |

### <a name="project-invoice-proposal-page-without-header-and-lines-views"></a>Тақырып және жол көріністері жоқ жоба есеп-шотының ұсыныс беті

| &nbsp; | &nbsp; |
|--------|--------|
| **Ұсынбау/жою себебі** | **Жобаның есеп-шот ұсынысы және есеп-шот журналының пішіндерін «Тақырып және жолдар» көрінісімен пайдаланыңыз** мүмкіндік кілтімен бірге енгізілген бетті жақсартуға байланысты ұсынылмаған. |
| **Басқа мүмкіндікпен ауыстырылды ма?** | Иә |
| **Өнімнің әсерге ұшыраған аймақтары** | Қолданба |
| **Орналастыру опциясы** | Өндірістік/қосалқы сценарийлерге арналған Project Operations; Ресурс/қосалқы емес сценарийлерге арналған Project Operations |
| **Күйі** | Ұсынылмаған: 2023 жылдың 1 наурызына қарай біз ескі (бұрынғы) бетті өшіреміз және әдепкі бойынша **Жобаның есеп-шот ұсынысы және есеп-шот журналының пішіндерін «Тақырып және жолдар» көрінісімен пайдаланыңыз** мүмкіндік кілтін қосамыз. |
