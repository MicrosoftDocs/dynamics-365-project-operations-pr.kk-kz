# <a name="use-the-schedule-board-to-book-project-resources"></a><span data-ttu-id="9a1ec-103">Жоба ресурстарына тапсырыс беру үшін кесте тақтасын пайдалану</span><span class="sxs-lookup"><span data-stu-id="9a1ec-103">Use the Schedule Board to book project resources</span></span>

[!include [banner](../includes/psa-now-project-operations.md)]

<span data-ttu-id="9a1ec-104">Ресурстарға жобадан жоба ішінде тапсырыс берумен қатар, кесте тақтасынан оларға біржола тапсырыс беруге немесе алдын ала тапсырыс беруге болады.</span><span class="sxs-lookup"><span data-stu-id="9a1ec-104">In addition to booking resources on a project from within a project, you can hard-book or soft-book resources from the Schedule Board.</span></span>

<span data-ttu-id="9a1ec-105">Кесте тақтасынан тапсырыс бермес бұрын, ресурс талаптарын жасау немесе құру керек.</span><span class="sxs-lookup"><span data-stu-id="9a1ec-105">Before you can book from the Schedule Board, you must create or generate resource requirements.</span></span> <span data-ttu-id="9a1ec-106">Кесте тақтасынан ресурс талаптарын жасау үшін келесі әрекеттерді орындаңыз.</span><span class="sxs-lookup"><span data-stu-id="9a1ec-106">Follow these steps to create resource requirements from the Schedule Board.</span></span>

1. <span data-ttu-id="9a1ec-107">Беттің төменгі жағындағы **тапсырыс беру талаптары** тақтасы жиылса, оны жаю үшін кеңейткіштің басқару элементін таңдаңыз.</span><span class="sxs-lookup"><span data-stu-id="9a1ec-107">If the **Booking Requirements** pane at the bottom of the page is collapsed, select the expander control to expand it.</span></span>
2. <span data-ttu-id="9a1ec-108">**Тапсырыс беру талаптары** тақтасындағы **Жоба** қойыншасынан тапсырыс беру қажет талапты таңдаңыз.</span><span class="sxs-lookup"><span data-stu-id="9a1ec-108">In the **Booking Requirements** pane, on the **Project** tab, select the requirement to book.</span></span>

    ![Жоба қойыншасында таңдалған талап](media/Resource-Management-image73.png)

3. <span data-ttu-id="9a1ec-110">Тапсырыс беруге болатын ресурстарға сүзгі қолдану және қолжетімді ресурстарды көру үшін **Қолжетімділігін табу** опциясын таңдаңыз.</span><span class="sxs-lookup"><span data-stu-id="9a1ec-110">Select **Find Availability** to filter the bookable resources and view the available resources.</span></span> 
4. <span data-ttu-id="9a1ec-111">Кесте тақтасынан бір немесе бірнеше ресурс таңдаңыз.</span><span class="sxs-lookup"><span data-stu-id="9a1ec-111">Select one or more resources from the Schedule Board.</span></span> 
5. <span data-ttu-id="9a1ec-112">Беттің оң жағындағы **Ресурсқа тапсырыс беруді жасау** тақтасына тапсырыс беру ақпаратын енгізіңіз, содан кейін **Тапсырыс беру және шығу** пәрменін таңдаңыз.</span><span class="sxs-lookup"><span data-stu-id="9a1ec-112">In the **Create Resource Booking** pane on the right side of the page, enter the booking information, and then select **Book and exit**.</span></span>

    ![Таңдалған тапсырыс беруге болатын ресурсқа арналған "ресурсқа тапсырыс беруді жасау" тақтасы](media/Resource-Management-image74.png)

6. <span data-ttu-id="9a1ec-114">**Ресурсқа тапсырыс беруді жасау** тақтасында талап таңдалған кезде, тапсырысты жасау үшін ресурстың бір немесе бірнеше ұяшығын таңдаңыз.</span><span class="sxs-lookup"><span data-stu-id="9a1ec-114">While the requirement is selected in the **Create Resource Booking** pane, select one or more cells of a resource to create the booking.</span></span>

    ![Ресурс үшін таңдалған бірнеше ұяшық](media/Resource-Management-image75.png)

7. <span data-ttu-id="9a1ec-116">**Тапсырыс беру** пәрменін таңдаңыз.</span><span class="sxs-lookup"><span data-stu-id="9a1ec-116">Select **Book**.</span></span>

<span data-ttu-id="9a1ec-117">Таңдалған ресурсты пайдалану арқылы талап орындалады.</span><span class="sxs-lookup"><span data-stu-id="9a1ec-117">The requirement is fulfilled by using the selected resource.</span></span> <span data-ttu-id="9a1ec-118">**Тапсырыс беру талаптары** тақтасында талаптың жаңартылғанын және ресурс "жобада тапсырыс берілген" ретінде көрсетілетінін байқауға болады.</span><span class="sxs-lookup"><span data-stu-id="9a1ec-118">In the **Booking Requirements** pane, notice that the requirement has been updated, and the resource is shown as booked on the project.</span></span>

![Жоба бойынша тапсырыс берілген ресурс](media/Resource-Management-image76.png)
